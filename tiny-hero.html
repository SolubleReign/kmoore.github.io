Enhancement Narratives

Software Engineering & Design

In this enhancement, I refactored the original monolithic game code into modular components, improving separation of concerns. I created distinct modules for player logic, enemy behavior, and game flow. This demonstrates my ability to design scalable and maintainable software systems.

Data Structures & Algorithms

I implemented an enemy journal system using dictionaries and lists to track defeated enemies, allowing sorting and lookup operations. This improved the game’s ability to manage dynamic data efficiently and demonstrated my understanding of structured data handling.

Databases

I integrated SQLite to persist player progress, inventory, and combat history. This replaced in-memory storage and allowed the game to maintain state across sessions, demonstrating database design and CRUD operations.

Professional Self-Assessment

Through CS-499, I have strengthened my ability to develop full-stack software systems, refactor existing codebases, and apply database integration techniques. I have also improved my ability to communicate technical decisions clearly and document software for professional presentation.

These experiences have prepared me for roles in software engineering, embedded systems, and backend development by demonstrating real-world application of computer science principles.
