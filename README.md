# Professional Self Assessment

  The coursework and creation of my ePortfolio have allowed me to showcase my strengths while demonstrating growth in software engineering, algorithms and data structures, databases, security, communication, and collaboration. The ePortfolio serves as a collection of projects that highlights both my technical skills and my ability to apply computer science concepts to real-world problems.

  One important skill I developed during the program was collaborating and communicating with stakeholders. Through project-based coursework, I learned how to analyze requirements, gather user needs, and translate those requirements into technical solutions. Many assignments required documenting system designs, explaining technical decisions, and presenting solutions in a way that could be understood by both technical and non-technical audiences. These experiences strengthened my communication skills and prepared me to work effectively within professional development teams.

  The program also enhanced my understanding of data structures and algorithms. Through various assignments, I learned how to evaluate efficiency, organize data effectively, and implement searching and sorting techniques to improve program performance. My software engineering skills were strengthened through projects that required modular design, object-oriented programming, testing, debugging, and code refactoring. Additionally, database coursework provided experience designing relational databases, writing SQL queries, and integrating persistent storage into applications. Security concepts were incorporated throughout the curriculum, emphasizing secure coding practices, input validation, authentication, and protecting data from common vulnerabilities.

  The artifacts included in this ePortfolio demonstrate the progression of these skills. Together, they showcase my ability to design, develop, enhance, and maintain software systems while applying computer science principles to solve complex problems. The Software Design and Engineering artifact highlights maintainability, modularity, and object-oriented programming practices. The Algorithms and Data Structures artifact demonstrates efficient data organization, searching, and sorting techniques. The Databases artifact illustrates database integration, persistent storage, and data management. Collectively, these artifacts represent the knowledge and skills I have gained throughout the program and demonstrate my readiness to begin a professional career in computer science.
